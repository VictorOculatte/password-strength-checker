# Password Strength Checker

Script educacional em Python que avalia a força de uma senha com base em critérios simples:
- comprimento (>=12)
- presença de letras maiúsculas
- presença de letras minúsculas
- presença de números
- presença de caracteres especiais

## Como usar
Rode:
python password_checker.py


## Aviso
Ferramenta apenas educacional. Não utilize em senhas de terceiros ou armazene senhas sensíveis neste repositório.
