# Uso - Password Strength Checker

## Requisitos
- Python 3

## Executar localmente
1. Abra um terminal na pasta do projeto.
2. Rode:
python password_checker.py
3. Insira a senha quando solicitado.
4. O script retorna uma classificação (Fraca / Média / Forte) e motivos para melhoria.

## Observações
- Ferramenta educacional: use apenas com senhas suas ou em ambiente controlado.
- Não armazena nem envia senhas para qualquer lugar.
